1. 查看所有分支
    `git branch`
2. 删除某一个分支
    `git branch -d 分支名称`
3. 强制删除某一个分支
    `git branch -D 分支名称`
4. 查看远程仓库地址
    `git remote -v`
5. 查看全局配置
    `git config -l`
6. 全局配置文件位置
    `.git/config`
7. 配置用户名和邮箱
    `git config --global user.name "John Doe"`
    `git config --global user.email johndoe@example.com`

1. 文本超出长度隐藏，显示3个点

    单行文本超出隐藏：需要手动指定`width`
    ```css
    overflow: hidden;
    text-overflow: ellipsis;
    white-space: nowrap;
    ```

    多行文本超出隐藏：需手动指定`width`和`height`
    ```css
    text-overflow: -o-ellipsis-lastline;
    overflow: hidden;
    text-overflow: ellipsis;
    display: -webkit-box;
    -webkit-line-clamp: 3; //行数需设置
    line-clamp: 3;
    -webkit-box-orient: vertical;
    ```
